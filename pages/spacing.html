Code

CSS Variables & Tailwind

Gunakan token langsung via CSS Custom Properties atau melalui Tailwind utility classes. Keduanya mengacu pada nilai yang sama.

styles.css — :root 
:root {
  /* ─── Spacing Scale (4px base) ───────────────── */
  --spacing-1:  0.25rem;   /* 4px  */
  --spacing-2:  0.5rem;    /* 8px  */
  --spacing-3:  0.75rem;   /* 12px */
  --spacing-4:  1rem;      /* 16px — default */
  --spacing-5:  1.25rem;   /* 20px */
  --spacing-6:  1.5rem;    /* 24px */
  --spacing-8:  2rem;      /* 32px */
  --spacing-10: 2.5rem;    /* 40px */
  --spacing-12: 3rem;      /* 48px */
  --spacing-16: 4rem;      /* 64px */
  --spacing-20: 5rem;      /* 80px */
  --spacing-24: 6rem;      /* 96px */
}
tailwind.config.js 
/** @type {import('tailwindcss').Config} */
module.exports = {
  theme: {
    extend: {
      spacing: {
        /* Maps Tailwind classes → CSS variables */
        'ds-1':  'var(--spacing-1)',   /* 4px  */
        'ds-2':  'var(--spacing-2)',   /* 8px  */
        'ds-3':  'var(--spacing-3)',   /* 12px */
        'ds-4':  'var(--spacing-4)',   /* 16px */
        'ds-6':  'var(--spacing-6)',   /* 24px */
        'ds-8':  'var(--spacing-8)',   /* 32px */
        'ds-12': 'var(--spacing-12)',  /* 48px */
        'ds-16': 'var(--spacing-16)',  /* 64px */
      },
    },
  },
}
HTML — Tailwind classes 
<!-- Padding all sides (16px) -->
<div class="p-4">Padded element</div>

Guidelines

Usage Guidelines

Do

  • Gunakan scale yang telah ditentukan untuk consistency
  • Mulai dengan spacing-4 (16px) untuk default padding/margin
  • Gunakan spacing-2 atau spacing-3 untuk compact components
  • Gunakan spacing-6 atau spacing-8 untuk section spacing
  • Maintain breathing room antara elemen

Don't

  • Jangan gunakan nilai spacing arbitrary (e.g. 7px, 13px)
  • Jangan campurkan unit yang berbeda (px, em, %) secara acak
  • Jangan abaikan hierarki spacing (kecil → dalam, besar → section)
  • Jangan gunakan hardcoded values bila token sudah tersedia
  • Jangan skip spacing step (misal langsung dari 8px ke 32px)
