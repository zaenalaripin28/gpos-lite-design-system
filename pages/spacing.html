Spacing System

12-step spacing scale dengan 4px base unit untuk consistent rhythm dan alignment

Spacing Scale

Semua spacing value berbasis 4px unit untuk mathematical harmony dan consistency.

--spacing-1: 4px

p-1, m-1, gap-1

--spacing-2: 8px

p-2, m-2, gap-2

--spacing-3: 12px

p-3, m-3, gap-3

--spacing-4: 16px

p-4, m-4, gap-4

--spacing-6: 24px

p-6, m-6, gap-6

--spacing-8: 32px

p-8, m-8, gap-8

--spacing-12: 48px

p-12, m-12, gap-12

--spacing-16: 64px

p-16, m-16, gap-16

Usage Guidelines

✅ Do

  • Gunakan scale yang telah ditentukan untuk consistency
  • Mulai dengan spacing-4 (16px) untuk default padding/margin
  • Gunakan spacing-2 atau 3 untuk compact components
  • Gunakan spacing-6 atau 8 untuk section spacing
  • Maintain breathing room antara elements

Code Examples

<!-- Padding all sides -->
<div class="p-4">Padded element</div>

<!-- Margin bottom -->
<p class="mb-4">Paragraph with bottom margin</p>

<!-- Gap between flex items -->
<div class="flex gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
