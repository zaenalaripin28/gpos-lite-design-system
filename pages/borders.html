Border Colors

Semantic Border Colors

/* Light border untuk subtle separation */
--color-border-light: var(--color-gray-200);

/* Primary border untuk borders */
--color-border: var(--color-gray-300);

/* Dark border untuk emphasis */
--color-border-dark: var(--color-gray-400);

✅ Usage Guidelines

  • border-light: Input fields, card separators
  • border: Default borders, dividers
  • border-dark: Focus states, emphasis
  • Gunakan semantic colors (success, error, warning) untuk status borders
