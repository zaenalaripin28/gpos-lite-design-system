Best Practices

Panduan penggunaan tipografi untuk memastikan konsistensi dan aksesibilitas di seluruh produk.

Do
  • Selalu gunakan Poppins sebagai font family utama untuk semua elemen UI.
  • Gunakan style dari typography scale yang telah ditentukan — jangan buat custom sizes.
  • Maintain font-size dan line-height yang sesuai untuk readability (minimum 12px untuk caption).
  • Gunakan semantic HTML tags (h1–h6, p, strong, em) untuk struktur konten.
  • Ensure contrast ratio memenuhi WCAG AA (4.5:1 untuk normal text, 3:1 untuk large text).
Don't
  • Jangan gunakan font family selain Poppins untuk elemen UI.
  • Jangan buat custom font sizes di luar typography scale yang ada.
  • Jangan gunakan font size kurang dari 12px untuk teks yang perlu dibaca.
  • Jangan mix font weights secara acak di luar ketentuan style yang ada.
  • Jangan skip line-height karena sangat memengaruhi readability dan aksesibilitas.
