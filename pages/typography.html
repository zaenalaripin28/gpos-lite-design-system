Code

CSS Variables & Tailwind

Gunakan token langsung via CSS Custom Properties atau melalui Tailwind utility classes. Keduanya mengacu pada nilai yang sama.

styles.css — :root 
:root {
  /* ─── Font Families ─────────────────────────────── */
  --font-primary: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
  --font-mono:    'JetBrains Mono', 'Courier New', monospace;

  /* ─── Font Sizes ─────────────────────────────────── */
  --font-size-xs:   0.75rem;    /* 12px */
  --font-size-sm:   0.875rem;   /* 14px */
  --font-size-base: 1rem;       /* 16px — default */
  --font-size-lg:   1.125rem;   /* 18px */
  --font-size-xl:   1.25rem;    /* 20px */
  --font-size-2xl:  1.5rem;     /* 24px */
  --font-size-3xl:  1.875rem;   /* 30px */
  --font-size-4xl:  2.25rem;    /* 36px */
  --font-size-5xl:  3rem;       /* 48px */

  /* ─── Font Weights ───────────────────────────────── */
  --font-weight-light:    300;
  --font-weight-normal:   400;
  --font-weight-medium:   500;
  --font-weight-semibold: 600;
  --font-weight-bold:     700;

  /* ─── Line Heights ───────────────────────────────── */
  --line-height-tight:   1.2;
  --line-height-snug:    1.375;
  --line-height-normal:  1.5;    /* default */
  --line-height-relaxed: 1.625;
  --line-height-loose:   2;
}
tailwind.config.js 
/** @type {import('tailwindcss').Config} */
module.exports = {
  theme: {
    extend: {
      fontFamily: {
        'sans': ['Poppins', '-apple-system', 'sans-serif'],
        'mono': ['JetBrains Mono', 'monospace'],
      },
      fontSize: {
        'ds-xs':   ['var(--font-size-xs)', { lineHeight: 'var(--line-height-normal)' }],
        'ds-sm':   ['var(--font-size-sm)', { lineHeight: 'var(--line-height-normal)' }],
        'ds-base': ['var(--font-size-base)', { lineHeight: 'var(--line-height-normal)' }],
        'ds-lg':   ['var(--font-size-lg)', { lineHeight: 'var(--line-height-normal)' }],
        'ds-xl':   ['var(--font-size-xl)', { lineHeight: 'var(--line-height-snug)' }],
        'ds-2xl':  ['var(--font-size-2xl)', { lineHeight: 'var(--line-height-tight)' }],
        'ds-3xl':  ['var(--font-size-3xl)', { lineHeight: 'var(--line-height-tight)' }],
        'ds-4xl':  ['var(--font-size-4xl)', { lineHeight: 'var(--line-height-tight)' }],
        'ds-5xl':  ['var(--font-size-5xl)', { lineHeight: '1' }],
      },
    },
  },
}
HTML — Typography usage examples 
<!-- H1: bold + tight line-height -->
<h1 class="text-4xl font-bold leading-tight text-gray-900">
  Page Title
</h1>

<!-- Body paragraph -->
<p class="text-base font-normal leading-normal text-gray-600">
  Body text yang nyaman dibaca di semua perangkat.
</p>

<!-- Label / badge -->
<span class="text-xs font-medium uppercase tracking-widest">
  Section Label
</span>

<!-- Inline code -->
<code class="font-mono text-sm text-red-600 bg-gray-100 px-1 rounded">
  const x = 42;
</code>

<!-- CSS Variable approach (vanilla CSS) -->
<p style="
  font-size: var(--font-size-base);
  font-weight: var(--font-weight-normal);
  line-height: var(--line-height-normal);
  color: var(--color-text-primary);
">Vanilla CSS typography</p>
Guidelines

Usage Guidelines

Do

  • Gunakan scale yang telah ditentukan, jangan buat custom sizes
  • Minimum 14px untuk body text agar mudah terbaca
  • Gunakan semantic HTML (h1–h6, p, strong, em, code)
  • Test readability di dark dan light background
  • Pastikan contrast ratio WCAG AA (4.5:1 untuk normal text)

Don't

  • Jangan buat custom font sizes di luar scale ini
  • Jangan gunakan font size kurang dari 12px
  • Jangan mix font families dalam satu component
  • Jangan skip line-height — selalu set secara eksplisit
  • Jangan gunakan lebih dari 2–3 font family berbeda
