Grid & Layout System

Responsive 12-column grid dengan breakpoints untuk flexible layouts

Responsive Breakpoints

Breakpoints untuk responsive design di berbagai ukuran layar.

XS (Extra Small)

--breakpoint-xs: 0

Mobile default

SM (Small)

--breakpoint-sm: 640px

Landscape mobile

MD (Medium)

--breakpoint-md: 768px

Tablet

LG (Large)

--breakpoint-lg: 1024px

Desktop

XL (Extra Large)

--breakpoint-xl: 1280px

Wide desktop

2XL (2XL)

--breakpoint-2xl: 1536px

Ultra wide desktop

Grid System

12-Column Grid

Gunakan Tailwind CSS grid utilities untuk layout yang flexible.

<!-- 2-column grid responsive -->
<div class="grid grid-cols-1 md:grid-cols-2 gap-4">
  <div>Column 1</div>
  <div>Column 2</div>
</div>

<!-- 3-column grid -->
<div class="grid grid-cols-1 md:grid-cols-3 gap-6">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
</div>

<!-- Custom span -->
<div class="grid grid-cols-12 gap-4">
  <div class="col-span-8">8 columns</div>
  <div class="col-span-4">4 columns</div>
</div>

Flex Layout

<!-- Flex row with gap -->
<div class="flex gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

<!-- Flex column -->
<div class="flex flex-col gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

<!-- Flex with justify content -->
<div class="flex justify-between items-center">
  <div>Left</div>
  <div>Right</div>
</div>

Layout Patterns

Mobile-First Approach

Mulai dari mobile design, tambahkan breakpoint untuk desktop:

<!-- Start mobile (1 column), then 2 cols on tablet, 3 cols on desktop -->
<div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3">
  <!-- Grid items -->
</div>

Container & Content Width

<!-- Max width container -->
<div class="max-w-6xl mx-auto px-4">
  <!-- Content -->
</div>

--max-w-xl: 36rem (576px)

Best Practices

✅ Do

  • Design mobile-first, enhance untuk larger screens
  • Gunakan responsive prefixes (sm:, md:, lg:, xl:)
  • Test layout pada semua breakpoints
  • Maintain consistent spacing dengan design system
  • Use semantic grid/flex utilities

❌ Don't

  • Jangan buat custom breakpoints
  • Jangan hardcode widths/heights
  • Jangan skip responsive testing
  • Jangan neglect touch targets (minimum 44x44px)
