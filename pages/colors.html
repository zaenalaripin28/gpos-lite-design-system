Color System

Comprehensive color palette dengan semantic naming untuk konsistensi visual

Pengenalan Color System

Sistem warna kami dibangun dengan 6 core color palettes dan grayscale yang komprehensif. Setiap warna memiliki 10 tingkat shade dari 50 (lightest) hingga 900 (darkest).

Struktur Naming

Format: --color-[palette]-[shade]

  • --color-primary-500 = Core primary color
  • --color-primary-100 = Light variant untuk backgrounds
  • --color-primary-900 = Dark variant untuk text/borders

✅ Usage Guidelines

  • Primary-500: Button primary, links, focus states
  • Primary-600: Hover states untuk primary buttons
  • Primary-100: Background untuk highlighted content
  • Primary-50: Subtle background untuk notifications

❌ Don't Use

  • Jangan mix primary color dengan semantic colors (success, warning, error)
  • Jangan gunakan primary-900 untuk backgrounds
  • Jangan buat custom shade dari primary color

Code Examples

Tailwind CSS:

<button class="bg-primary-500 text-white hover:bg-primary-600">
  Primary Button
</button>

CSS Variables:

.button-primary {
  background-color: var(--color-primary-500);
  color: white;
}

.button-primary:hover {
  background-color: var(--color-primary-600);
}

Accessibility & Contrast

WCAG 2.1 AA Compliance

Semua kombinasi color pair dalam design system telah memenuhi WCAG 2.1 AA standards untuk contrast ratio (minimum 4.5:1 untuk normal text, 3:1 untuk large text).

✅ Approved Combinations

  • Primary-500 text on Gray-50 background ✓ 4.87:1
  • Gray-700 text on Gray-50 background ✓ 7.21:1
  • Gray-900 text on White background ✓ 19.6:1
  • White text on Primary-600 background ✓ 6.41:1

❌ Not Recommended

  • Gray-400 text on Gray-100 background - insufficient contrast
  • Primary-100 text on Gray-50 background - too light
  • Color alone to convey status - selalu tambahkan icons atau text
